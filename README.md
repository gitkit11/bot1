# Chimera AI v4.3 — Telegram-бот для прогнозирования ставок

**Chimera AI** — профессиональный Telegram-бот для анализа футбольных матчей. Использует математические модели, три независимых AI агента и реальные xG данные для поиска ставок с положительным EV.

## Архитектура v4.3

| Компонент | Технология | Вес в ансамбле | Описание |
|-----------|-----------|---------------|---------|
| **Пуассон + xG** | math_model.py + Dixon-Coles | 40% | xG данные через Understat, поправка Dixon-Coles для малых счётов |
| **ELO + Форма** | math_model.py | 30% | Динамические рейтинги по 1732 матчам 5 лиг + бонус формы последних 5 матчей |
| **GPT-4o (Маэстро)** | OpenAI API | 15% (AI блок) | Главный арбитр, Kelly criterion, Value Bets |
| **Llama 3.3 70B** | Groq API | 15% (AI блок) | Независимый прогноз, тотал, обе забьют |
| **Gemma2-9b-it** | Groq API | 15% (AI блок) | Тактический анализ (3-й агент) |
| **Букмекеры** | The Odds API | 10% | Рыночная мудрость, нормализованные вероятности |
| **Пророк (LSTM)** | TensorFlow | 5% | Нейросеть на 66k матчей, исторические паттерны |

### Дополнительные модули
- **Оракул** — DistilBERT анализ новостей + GNews API (настроение команды)
- **Травмы** — автопоиск через GNews + GPT-4o-mini извлечение имён (параллельно для обеих команд)
- **Value Bets** — EV > 10%, коэф >= 1.55, уверенность >= 52% (профессиональные пороги)
- **ROI трекер** — SQLite база с полной историей прогнозов, точностью по каждой модели, ROI

## Установка

### 1. Клонировать репозиторий
```bash
git clone <URL>
cd chimera_bot
```

### 2. Создать .env файл
```bash
cp .env.example .env
# Заполнить своими ключами в .env
```

### 3. Установить зависимости
```bash
pip install -r requirements.txt
```

### 4. Запустить
```bash
python main.py
```

## Команды бота

| Команда | Описание |
|---------|---------|
| `/start` | Главное меню |
| `/results` | Статистика прогнозов: точность, ROI, Value Bets |
| Анализ матча | Полный анализ с Пуассоном, ELO, AI агентами, травмами |

## Автоматические задачи

- **Каждый час** — проверка результатов завершённых матчей, обновление ROI
- **Каждый понедельник 3:00** — перекалибровка ELO по свежим результатам сезона

## Структура файлов

```
main.py              — Telegram бот, основной цикл
agents.py            — GPT-4o, Llama, Gemma2 агенты + ансамбль
math_model.py        — Пуассон (Dixon-Coles), ELO, форма
maestro_ai.py        — Маэстро (Value Bets, Kelly criterion)
oracle_ai.py         — Оракул (новостной анализ)
understat_stats.py   — xG статистика (Understat + fallback api_football)
injuries.py          — Травмы и дисквалификации (async)
database.py          — SQLite: прогнозы, результаты, ROI
elo_calibrate.py     — Ручная перекалибровка ELO
api_football.py      — Статистика через API-Football
config.py            — Конфигурация (читает .env)
.env                 — API ключи (НЕ коммитить!)
.env.example         — Шаблон для .env
elo_ratings.json     — Текущие ELO рейтинги (96 команд, 5 лиг)
team_form.json       — Форма последних 5 матчей
```
