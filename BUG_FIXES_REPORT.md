### 2. **Llama 403 Forbidden Error** (Строки 107-157 в логах)
**Причина:** Groq API возвращает 403 "Access denied". Возможные причины:
- Неправильный API ключ
- Проблема с сетью/регионом
- Лимит запросов исчерпан

**Решение:**
- ✅ Переписан `agents.py` с улучшенной обработкой ошибок
- ✅ Добавлен автоматический **fallback на GPT-4o** при ошибке Llama
- ✅ Улучшена обработка исключений (PermissionDeniedError, timeout и т.д.)
- **Результат:** Если Llama недоступна, бот автоматически использует GPT, и анализ продолжается без перерыва

---

### 3. **Playwright Sync API inside asyncio loop** (Строки 101-105 в логах)
**Причина:** В `hltv_scraper.py` используется синхронный Playwright внутри асинхронного цикла main.py, что вызывает конфликт.

**Решение:**
- ✅ Создан новый синхронный модуль `sports/cs2/hltv_sync.py` (без asyncio)
- ✅ Обновлен импорт в `main.py` (строка 1955): теперь используется `hltv_sync` вместо `hltv_stats`
- ✅ Модуль работает с кэшем и fallback на статические данные
- **Результат:** Нет больше конфликтов asyncio, CS2 анализ работает стабильно
